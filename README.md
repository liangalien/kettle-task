### Kettle在线任务调度平台

1、支持上传、下载ktr/kjb文件
2、支持手动执行、定时执行
3、支持日志实时查看

后端springboot

```
mvn install
```


前端vue、element-ui
```
npm i
npm run dev
npm run build
```
